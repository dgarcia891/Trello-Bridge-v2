Privacy Policy for Trello Bridge

Last updated: September 13, 2025
Version: 1.0.0

Overview

Trello Bridge is a Chrome extension that helps users manage Trello cards while browsing the web. This privacy policy explains how we handle your data.

Data Collection

We do not collect, store, or transmit any personal data to our servers.

Data Storage

  • Local Storage: The extension stores your Trello API credentials and preferences locally on your device using Chrome's storage API
  • Trello API: All Trello data (boards, cards, lists) is accessed directly through Trello's official API using your credentials
  • No Third-Party Services: We do not use any third-party analytics, tracking, or data collection services

What We Store Locally

  • Trello API key and token (encrypted by Chrome)
  • Selected board preferences
  • Auto-check domain settings
  • UI preferences and settings
  • Last used board/list for Quick Create
  • Search preferences and board selections

Data Transmission

  • Trello API: The extension communicates directly with Trello's servers using your API credentials
  • No External Servers: We do not have any servers that receive or process your data
  • Secure Communication: All API calls use HTTPS encryption

Features and Data Usage

  • Card Search: Searches your Trello boards using page URLs and titles
  • Card Editing: Modifies existing Trello cards with your changes
  • Card Creation: Creates new Trello cards with page information
  • Manual Search: Allows custom text searches within your Trello boards
  • Board Selection: Remembers your preferred boards for searches
  • Domain Auto-Check: Stores domain preferences for automatic searching

Permissions Explained

  • activeTab: Access to current tab information for card creation and search
  • storage: Save your preferences and settings locally
  • tabs: Get tab information for creating cards with page details
  • scripting: Execute content scripts when needed
  • https://api.trello.com/*: Access Trello's official API

Data Security

  • All data is stored locally on your device
  • Trello API credentials are encrypted by Chrome's storage system
  • No data leaves your device except for direct Trello API calls
  • We do not have access to your Trello data or credentials
  • Input validation prevents security issues

Third-Party Services

  • Trello: We integrate with Trello's official API using your credentials
  • Chrome Extensions: The extension runs within Chrome's secure extension environment

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the version number and date above.

Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store listing.

Your Rights

Since we don't collect personal data, there's no personal information to access, modify, or delete. All data is stored locally on your device and can be removed by uninstalling the extension.
