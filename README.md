# Privacy Policy Hosting for Trello Bridge

Privacy policy page for the Trello Bridge Chrome extension, hosted via GitHub Pages.

**Live URL:** https://dgarcia891.github.io/Trello-Bridge-v2/

## Contents

- `index.html` - Privacy policy page

## Usage

GitHub Pages automatically deploys from the main branch. To update the privacy policy, edit `index.html` and commit the changes.
